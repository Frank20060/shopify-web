# Guía de Desarrollo con Shopify

## Introducción

Esta es una guía completa para desarrolladores que quieren trabajar con la plataforma Shopify.

### Temas principales

- **Liquid**: El lenguaje de plantillas de Shopify
- **Shopify Functions**: Lógica backend con Rust
- **APIs**: Integración con servicios externos

### Primeros pasos

1. Instala las herramientas necesarias
2. Configura tu tienda de desarrollo
3. Empieza con temas básicos

```liquid
{% if product.available %}
  <p>Producto disponible</p>
{% endif %}
```

¡Feliz desarrollo! 🚀
